# Getting Started

> **Note:** Currently `Solias` is in beta and only Bar chart is available and more chart types will be added in the future.

Install **Solias** in your project

```sh
npm i solias
```
> You can use `Yarn` as well.

### Integration

For integrating **Solias** in your project, refer the [official Documentation](https://stenciljs.com/docs/overview) from **StencilJS** for each framework.

* [Angular](https://stenciljs.com/docs/angular)
* [React](https://stenciljs.com/docs/react)
* [Vue](https://stenciljs.com/docs/vue)
* [JavaScript](https://stenciljs.com/docs/javascript)
* [Ember](https://stenciljs.com/docs/ember)
